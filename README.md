## Getting Started

Install the module with: `npm install bartndr-web`

```js
var bartndr-web = require('bartndr-web');
bartndr-web.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g bartndr-web
$ bartndr-web --help
$ bartndr-web --version
```


```sh
# creates a browser.js
$ grunt browserify
```
